# Airbnb Listings 2016 - Data Analysis and Visualization Using Tableau

## Project Overview

This project involves the analysis and visualization of the Airbnb Listings 2016 dataset, which contains various details about Airbnb listings worldwide. The goal was to explore the dataset, extract key insights, and present them in an interactive and visually appealing manner using Tableau.

## Dataset

The dataset used in this project can be found on [Kaggle](https://www.kaggle.com/datasets/alexanderfreberg/airbnb-listings-2016-dataset). It includes various features such as listing prices, locations, property types, and other relevant information.

## Tools and Technologies

- **Tableau**: For data visualization and dashboard creation.
- **Microsoft Excel**: Initial data cleaning and preprocessing.

## Key Insights

The analysis focused on several key areas:

- **Geographical Distribution**: Visualization of listings across different regions.
- **Price Analysis**: Distribution of prices across different locations.
- **Revenue Per Year**: Visualization of price per calendar for year 2016.
- **Price Per Number of Bedrooms**: Visualization of the price distribution based on the number of bedrooms for different type of rooms.

## Visualizations

The Tableau dashboards created in this project include:

1. **Global Distribution Map**: A latitude and longitude map showing the average listing price for different zipcode in USA.
2. **Price Distribution Chart**: A bar chart depicting the range of prices across different regions.
3. **Revenue Per Year**: A line plot of price per weeks for year 2016.
4. **Price and Count of Number of Bedrooms**: A bar chart and a table for different number of bedrooms for room on the basis of price.
